Synopsis

link file1 file2

Description

The link creates a hard link, file2, which will refer to the same file as file1.

Operands

file1

The pathname of an existing file.

file2

The pathname of the new directory entry which will be created.

Diagnostics

The following error values are returned:

Error value: 0
Diagnostic: Successful completion.
Error value: >0
Diagnostic: An error occurred.

Standards

The link utility conforms to the Single UNIX Specification, Version 3 (SUSv3), and the X/Open System Interfaces Extension (XSI).
